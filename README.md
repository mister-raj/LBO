# Leveraged Buyout (LBO) Model – Indian Healthcare Case

This repository contains a **leveraged buyout (LBO) financial model** built to evaluate the acquisition of an Indian hospital platform under a sponsor-led buyout structure.

The model analyzes transaction feasibility, capital structure, debt repayment, and equity investor returns over a 5-year holding period.

---

## Model Overview

The case assumes a private equity fund acquires 100% of the target company using a leveraged capital structure and exits at an EBITDA multiple.

### Transaction Structure

* **Entry Multiple:** 18.0× EBITDA
* **Exit Multiple:** 18.0× EBITDA
* **Holding Period:** 5 Years
* **Existing Net Debt:** ₹2,800 Cr
* **Transaction Fees:** ₹180 Cr

---

## Operating Assumptions (LTM Base)

* **Revenue:** ₹8,500 Cr
* **Operating Costs:** ₹5,100 Cr
* **SG&A / OpEx:** ₹1,700 Cr
* **D&A:** ₹450 Cr
* **Capex:** ₹750 Cr
* **Net Working Capital:** ₹600 Cr
* **Revenue Growth:** 11% CAGR
* **Tax Rate:** 25%

Margins and cost structures are held constant across projections.

---

## Financing Structure

| Debt Tranche | Amount    | Interest |
| ------------ | --------- | -------- |
| Revolver     | ₹1,000 Cr | 8.5%     |
| Term Loan    | ₹6,500 Cr | 10.25%   |
| Senior Notes | ₹3,500 Cr | 12.75%   |

Remaining funding is sponsor equity.

---

## What the Model Calculates

* Sources & Uses of funds
* Entry valuation & capital structure
* 5-year operating projections
* Free cash flow generation
* Debt amortization & cash sweep
* Exit equity value
* Sponsor IRR & MOIC
